# Job Application Tracker - Notion Template

## How to Create This Template in Notion (then sell it on Gumroad)

### Step 1: Create a New Notion Page
- Title: "Job Application Tracker 2026"
- Add an icon and cover image for visual appeal

### Step 2: Create the Main Database
Create a **Full Page Database** with these properties:

| Property | Type | Options |
|----------|------|---------|
| Company | Title | - |
| Position | Text | - |
| Status | Select | Applied, Phone Screen, Technical Interview, Onsite, Offer, Rejected, Withdrawn |
| Priority | Select | High, Medium, Low |
| Salary Range | Text | - |
| Location | Text | - |
| Remote? | Checkbox | - |
| Applied Date | Date | - |
| Last Contact | Date | - |
| Next Step | Text | - |
| Referral | Text | - |
| Job URL | URL | - |
| Resume Version | Select | v1, v2, v3 |
| Cover Letter | Checkbox | - |
| Notes | Text | - |
| Excitement (1-5) | Number | - |

### Step 3: Create Views

**View 1: Pipeline Board (Kanban)**
- Group by: Status
- Sort by: Applied Date (Newest first)
- This gives a visual pipeline of all applications

**View 2: All Applications (Table)**
- Show all properties
- Sort by: Applied Date (Newest first)
- Filter: Status is not "Rejected" and not "Withdrawn"

**View 3: Follow-Up Needed (Table)**
- Filter: Last Contact is before 7 days ago AND Status is not "Rejected"
- Sort by: Last Contact (Oldest first)

**View 4: Calendar View**
- Based on: Applied Date
- Shows application timeline

**View 5: By Priority (Board)**
- Group by: Priority
- Good for focusing on high-priority applications

### Step 4: Add Dashboard Section (above database)
Add these callout blocks above the database:

```
📊 Quick Stats
- Total Applications: [Count all]
- Active Pipeline: [Count where Status != Rejected/Withdrawn]
- Response Rate: [Phone Screens / Total Applied]
- Offer Rate: [Offers / Total Applied]
```

### Step 5: Add Template Buttons
Create database templates for quick entry:
- "Quick Apply" - Pre-fills Status=Applied, Date=Today
- "Referral Application" - Pre-fills with referral fields highlighted

### Step 6: Add Supporting Pages

**Interview Prep Page**
- Common behavioral questions list
- STAR method template
- Company research template
- Technical prep checklist

**Networking Tracker (Linked Database)**
- Contact Name, Company, Last Contacted, Notes, LinkedIn URL

**Offer Comparison Table**
- Company, Base Salary, Bonus, RSUs, Benefits, PTO, Remote Policy, Total Comp
