## Step-by-Step Launch Plan

### Week 1: Set Up Your Storefronts

**Gumroad (free, no monthly fee, takes 10% per sale)**
1. Go to gumroad.com and create an account
2. Upload each product with:
   - 3-5 screenshots/mockups
   - Compelling title with keywords
   - Detailed description highlighting benefits
   - Set price with a "pay what you want" minimum

**Etsy (for resume + spreadsheet templates)**
1. Open an Etsy shop (one-time $15 setup, $0.20 per listing)
2. List the resume template and budget tracker
3. Use keyword-rich titles (Etsy SEO is crucial)
4. Create mockup images using Canva (free)

### Week 2: Deploy the Web Tool

**For the JSON Formatter:**
1. Get a domain (e.g., devtoolkit.io) - ~$10/year on Namecheap
2. Host on Vercel or Netlify (free)
3. Apply for Google AdSense
4. Submit to Google Search Console for indexing

### Week 3: Build the Notion Template
1. Follow the guide in Notion_Job_Tracker_Template_Guide.md
2. Create it in your Notion workspace
3. Take screenshots of all views
4. List on Gumroad

### Week 4+: Scale and Expand
- Build more templates (cover letter, project manager resume, etc.)
- Build more web tools (CSS formatter, Base64 encoder, regex tester)
- Create bundle deals (resume + cover letter template = $14.99)
- Share on relevant Reddit communities and social media

## Important Reminders

1. **Consult an immigration attorney** before listing products for sale
2. **Keep it passive** - build it, list it, let the marketplace handle sales
3. **Don't actively market as a "business"** - these are creative digital products
4. **Report all income on your taxes** - even passive income is taxable
5. **Start small, iterate** - your first product might not sell much, but your 10th will
